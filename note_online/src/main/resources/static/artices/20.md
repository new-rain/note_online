| Ctrl+数字1-6（字母上方，小键盘不行） | 格式化为标题（标题共有6个等级）                              |

| Ctrl+b                               | 文字加粗                                                     |
| Ctrl+c                               | 复制                                                         |

| Ctrl+f                               | 查找                                                         |
| Ctrl+g                               |                                                              |

| Ctrl+i                               | 文字斜体                                                     |
| Ctrl+j                               | 滑动内容，使光标所在行位于页面上方（第三行左右）             |
| Ctrl+k                               | 创建链接，如光标处有内容，使内容转换为超链接（需要自己填写url） |
| Ctrl+l                               | 选中光标所在位置的一句内容（。？！结尾）                     |
| Ctrl+m                               |                                                              |

| Ctrl+w                               |                                                              |
| Ctrl+x                               | 剪切                                                         |
| Ctrl+y                               | 回退撤销                                                     |
